Initial TPD Procedure
H on Si(100)
2010.07.13 10:19
1. Set-up
1.1. Verify that TPD_Controller.vi is communicating properly with instruments (only on first run since program was started, power supply leads should be disconnected).
1.1.1. Lakeshore temperature controller
1.1.2. Kepco power supply
1.1.3. Mikron pyrometer
1.2. Put sample in elevator.
1.3. Outgas W filament.
1.3.1. Filament position = 4.0cm
1.3.2. Turn Variac to 10% for 5 min
1.3.3. Turn Variac to 2%
1.3.4. Filament position = 0cm.
1.4. Move sample to prep chamber stage.
2. TPD on bare Si(100) with clean chamber.
2.1. Input controller parameters/metadata.
2.1.1. Metadata (all self-explanatory)
2.1.2. Maximum values
2.1.2.1. Maximum Current = 4 A
2.1.2.2. Maximum voltage = 20 V
2.1.2.3. Maximum temperature = 800 C
2.1.3. Temperature Schedule
2.1.3.1. 350C, 0s
2.1.3.2. 350C, 50s
2.1.3.3. 650C, 200s
2.1.4. Gain Schedule
2.1.4.1. Kc = .003, Ti = .1, Td = .001, max Temp = 0C
2.1.4.2. PID calculates current
2.2. Flash sample
2.2.1. Heat sample to 1250 C for 60 s.
2.2.1.1. Use Kepco power supply.
2.2.1.2. Increase current to ~9 A, adjusting to attain final temperature.
2.2.1.3. Measure temperature with HT Pyrometer.
2.2.1.4. Keep an eye on thermocouple temperature, note if and when thermocouple breaks.
2.3. Begin recording RGA data.
2.4. Start temperature control program.
2.5. Save temperature and RGA data after experiment finishes.
3. TPD on clean Si(100) with chamber exposed to atomic H.
3.1. Input controller parameters/metadata.
3.1.1. Metadata (all self-explanatory)
3.1.2. Maximum values
3.1.2.1. Maximum Current = 4 A
3.1.2.2. Maximum voltage = 20 V
3.1.2.3. Maximum temperature = 800 C
3.1.3. Temperature Schedule
3.1.3.1. 350C, 0s
3.1.3.2. 350C, 50s
3.1.3.3. 650C, 200s
3.1.4. Gain Schedule
3.1.4.1. Kc = .003, Ti = .1, Td = .001, max Temp = 0C
3.1.4.2. PID calculates current
3.2. Flash sample.
3.2.1. Heat sample to 1250 C for 60 s.
3.2.1.1. Use Kepco power supply.
3.2.1.2. Increase current to ~9 A, adjusting to attain final temperature.
3.2.1.3. Measure temperature with HT Pyrometer.
3.2.1.4. Keep an eye on thermocouple temperature, note if and when thermocouple breaks.
3.3. Put sample in elevator.
3.4. Expose chamber to atomic H.
3.4.1. Filament position = 4.0cm
3.4.2. Begin recording RGA data.
3.4.3. Veriac at 4%
3.4.4. Flow H2 into chamber.
3.4.4.1. 2*10-6 torr for 4 minutes
3.4.5. Veriac at 2%
3.4.6. Filament position = 0cm
3.4.7. Save RGA data.
3.5. Put sample on prep chamber stage.
3.6. Start temperature control program.
3.7. Begin recording RGA data.
3.8. Save temperature and RGA data after experiment finishes.
